# Privacy Policy

This Privacy Policy explains how Bader Content Bot handles information.

## Data Collection
The application does not collect, store, or share any personal user data.

## Usage
Bader Content Bot automatically generates and publishes educational and moral short videos to a single TikTok account using the TikTok Content Posting API.

## Third-Party Services
The app uses TikTok APIs only for publishing content and does not access user private information.
